# AstroMD Web

占星術のホロスコープデータをMarkdown形式で生成するWebツール、[Astromd](https://astro-md-three.vercel.app/)のソースコードです。  
生成されたデータは、ChatGPTやClaude、GeminiなどのAI（LLM）が解釈しやすいように最適化されています。

## ✨ 主な機能

- **ホロスコープのMarkdown生成**: 出生情報（日時と場所）から、天体と感受点の度数、サビアンシンボル、アスペクト、ハウス情報をMarkdownテーブルとして出力します。
- **AIとの連携に最適化**: 生成されたMarkdownをコピーしてAIに渡すだけで、詳細なホロスコープリーディングを依頼できます。
- **シングル・ダブルチャート対応**: 個人のネイタルチャート（単体）と、シナストリーやトランジット分析に使えるダブルチャート（二重円）の両方に対応しています。画像は生成されません。
- **時刻不明でも利用可能**: 出生時刻が不明な場合でも、正午生まれとしてハウスや感受点を除いた基本的なチャートを生成できます。
- **マイナーアスペクトの選択**: 必要に応じてマイナーアスペクトの表示/非表示を切り替えられます。

## 🛠️ 使用技術

- **バックエンド**: Flask (Python)
- **天文計算**: pyswisseph (Swiss Ephemeris)
- **ジオコーディング**: Nominatim / OpenStreetMap
- **ホスティング**: Vercel
